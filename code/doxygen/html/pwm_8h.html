void Init_PWM ( void  )

Initialize PWM.

Sets up the PWM peripheral. The PWM has a pre-scalar of 10kHz and a counter rate of 100Hz. Because the PWM signal is used to control the sensor heater, it is initialized to 0% duty cycle.

None
